# University Lost and Found Management System

## Overview
This project is a **full-stack web application** developed for the University of Moratuwa to digitize and streamline the process of reporting, tracking, and recovering lost items on campus.

The system provides a centralized platform for:
- Reporting lost items
- Submitting found items
- Automatic matching between lost and found items
- Notifications and reward system
- Administrative control and monitoring

---

## Tech Stack
- **Frontend (User + Admin):** React.js
- **Backend:** Spring Boot
- **Architecture:** Full-stack web application
