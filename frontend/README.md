# User Frontend

## Overview
This module contains the **user-facing frontend** of the Lost and Found Management System.

It allows students and staff to interact with the system.

---

## Responsibilities
- User interface for reporting items
- Viewing lost and found items
- Viewing matches
- Managing user profile and history

---

## Features
- User registration and login
- Report lost items
- Report found items
- View matches
- Notification system
- Profile and reward points display

---

## Tech Stack
- React.js
- REST API integration with backend (Spring Boot)

---

## Notes
- This is a **frontend-only module**
- Backend handles all logic and data processing
