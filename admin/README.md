# Admin Dashboard (Frontend)

## Overview
This module contains the **admin-facing frontend** of the Lost and Found Management System.

It provides administrative tools for managing users, monitoring activity, and moderating content.

---

## Responsibilities
- Admin dashboard UI
- User management interface
- Item moderation interface
- System analytics display
- Audit logs visualization

---

## Features
- View all registered users
- Ban / unban users
- Promote / demote roles
- View and delete item reports
- View system statistics (users, reports, matches)
- Access audit logs

---

## Tech Stack
- React.js
- REST API integration with backend (Spring Boot)

---

## Notes
- This is a **frontend-only module**
- All business logic and data handling are managed by the backend
